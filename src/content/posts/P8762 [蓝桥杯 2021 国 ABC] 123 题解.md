## 前置知识:求和公式 (（首项 + 末项） × 项数 ÷ 2)
### 思想:前缀和+二分

因为数组太大，所以可以用前缀和。

所以题意可化为:

![](https://cdn.luogu.com.cn/upload/image_hosting/7idcn4fs.png?x-oss-process=image/resize,m_lfit,h_170,w_225)

我们可以发现 : **每一行的和=在原数列中最后的一个位置**。

对于原数列第$j$个数，可以用二分查找找每行第$i$个数比它小，第$i+1$个数比它大 所以第$j$个数在第$i+1$行。


```cpp
#include <iostream>
#include <iostream>
#define int long long
using namespace std;
const int MAXN = 1e7+10;
int pre[MAXN];
inline int sum(int i) { //求和公式 
	return (1 + i) * i / 2;
}
inline int find(int k) {
    int l = 1, r = MAXN;
    while(l <= r) {
        int mid = (l + r)/2;
        if(sum(mid) < k) l = mid + 1;
        else  r = mid - 1;
    }
    return pre[r] + sum(k - sum(r));
}
int T,len;
signed main() {
    cin >> T;
    for(int i = 1; len <= 1e12+10; i++) {
        len += i;
        pre[i] = pre[i-1] + sum(i);//前缀和 
    }
    while(T--) {
        int l, r;
        cin >> l >> r;
        cout << find(r) - find(l - 1) << endl;
    }
    return 0;
}
```
